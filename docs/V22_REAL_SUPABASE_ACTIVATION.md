# V22 Roadmap — Real Supabase Activation

## V22 цел
Да вземем реални Supabase keys и да активираме DATA_MODE=supabase.

## Стъпки
1. Създай Supabase project.
2. Run cloud/supabase/schema.sql.
3. Вземи URL + keys.
4. Сложи ги в .env.
5. Пусни supabase check.
6. Switch DATA_MODE=supabase.
7. Restart server.
8. Провери activeDataMode=supabase.
9. Направи тестов профил.
10. Провери таблиците в Supabase.
