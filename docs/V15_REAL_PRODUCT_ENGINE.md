# V15 Real Product Engine

EchoProfile вече работи като малка продуктова система.

## Клиентски поток

1. Клиентът прави Mini Echo.
2. Вижда базов доклад.
3. Натиска заявка за плащане.
4. Заявката влиза в Admin Engine.
5. Админът потвърждава плащане.
6. Системата генерира еднократен код.
7. Клиентът отключва Full Echo.
8. Кодът става използван.

## Admin поток

- Вход с PIN
- Преглед на профили
- Преглед на плащания
- Потвърждаване на плащане
- Генериране на кодове
- Метрики
- Export
- Events log

- cloud adapter
