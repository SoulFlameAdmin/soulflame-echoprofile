# V21 Admin Security

## Какво направи V21
- Смени default Admin PIN.
- Записа новия PIN в private/v21-admin-pin.txt.
- Добави private/ в .gitignore и .vercelignore.
- Подготви Supabase key intake.
- Добави security pages и scripts.

## Новият Admin PIN
Виж:

private/v21-admin-pin.txt

## Преди Vercel
Не качвай:
- .env
- private/
- data/
- backups/
- exports/
